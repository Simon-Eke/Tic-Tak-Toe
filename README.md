# Tic-Tak-Toe

# - Börja med att skapa en tom bräda med en 3x3-rutnät.
# - Implementera funktioner för spelarens och datorns drag.
# - Skriv kod för att kontrollera om någon har vunnit genom att kolla efter tre markörer i rad.

# Detaljerade tips:

# - Du kan använda en tvådimensionell lista eller array för att representera brädet.
# - Använd loopar och villkor för att hantera användarens och datorns drag.
# - För att kontrollera vinst, kan du använda villkorssatser för att kolla efter tre markörer i rad i horisontell, vertikal och diagonal riktning.

# Lycka till med övningen! 

## Extra utmaning

# - Gör så att spelaren får välja som den spelar med X eller O
# - Gör så att spelaren får välja storlek på brädet. Bygg så att spelet kan hantera det och räkna ut vinst oavsett hur stort brädet är.
# - Gör så att spelaren kan välja hur många markörer i rad som behövs för att vinna spelet
